
To run this code, download the Spotify Sorter folder and run the main.exe executable.
Follow the prompts in the terminal.
Once you see the "End of program." print, you know the program has finished running.
To rerun the sorter simply rerun the executable file.

If for some reason main.exe doesn't run, here are directions to recompile it.
To compile this code, download the Spotify Sorter folder and open it in a terminal. 
Run the following compile command:
    g++ -o main.exe .\src.\main.cpp
This shoulde create a main.exe in the Spotify Sorter folder, NOT in the src folder.
Then run the executable main.exe file.
